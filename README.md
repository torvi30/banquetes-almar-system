# Banquetes Almar System

Sistema web para gestión de eventos y cotizaciones para Banquetes Almar.

## Tecnologías
- Node.js
- Express
- MySQL
- HTML
- CSS
- JavaScript

## Estructura del proyecto

backend/
- API REST
- autenticación admin
- conexión a base de datos

frontend/
- página pública
- formulario de cotización
- panel administrador

database/
- script SQL de la base de datos

## Instalación

1. Clonar el repositorio

git clone https://github.com/torvi30/banquetes-almar-system.git

2. Instalar dependencias

cd backend  
npm install

3. Ejecutar servidor

node server.js

Servidor disponible en:

http://localhost:3001
